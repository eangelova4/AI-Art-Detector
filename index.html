What this tool does
This website uses a Google Teachable Machine image classifier to help determine whether a piece of artwork is more likely to be AI-generated or human-made. All processing happens locally in your browser.

How to use it

  1. Upload an image or enable your webcam.
  2. The model analyzes the image locally.
  3. View the probability and final result.
